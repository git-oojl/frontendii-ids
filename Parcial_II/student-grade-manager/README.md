# Gestor de calificaciones de estudiantes

Aplicación de React creada para la Práctica 4. Permite agregar, mostrar, filtrar y eliminar estudiantes usando componentes, props y `useState`.

## Funciones principales

- Agregar estudiantes con nombre, curso y calificación.
- Validar que el nombre no esté vacío.
- Validar que la calificación esté entre 0 y 100.
- Mostrar el estado del estudiante como aprobado o reprobado.
- Eliminar estudiantes con `filter()`.
- Mostrar total de estudiantes y promedio general del grupo.
- Filtrar por todos, aprobados y reprobados.

## Componentes

- `App`
- `StudentForm`
- `StudentList`
- `StudentCard`

## Comandos

```bash
npm install
npm run dev
```
